# TOY'S HOUSE

## Descripción
Repositorio creado con la finalidad de explicar el desarrollo de mipagina web dedicado a la 
venta de juguetes de manera online.

## Características Principales
- Fácil de usar
- Colaborativo

## Tecnologías
### Frontend
- HTML5, CSS3, Boostrap

### Backend 
- Node.js / Javascript / Express
- Base de datos: PosgresSQL

## Estructura del Proyecto
Se consideró la estructura del frontend y backend en el mismo repositorio ya que el proyecto trabajado 
es de caracter individual, archivosde procesos en la raiz del directorio, asignación de frontend y backend 
en sus respectivas carpetas.
